Q1. Explain the usage of ternary operator with syntax and write a program to check whether the number is even or odd using the ternary operator.

Usage of the Ternary Operator in JavaScript :-

The ternary operator in JavaScript is a concise way to perform conditional operations. It is also known as the conditional operator. The ternary operator takes three operands, hence the name "ternary." It is often used as a shortcut for simple if-else statements.
  • Syntax:- condition ? expressionIfTrue : expressionIfFalse;

  • condition: This is the condition to be evaluated. If the condition is true, the operator returns the value of expressionIfTrue; otherwise, it returns the value of expressionIfFalse.
  • expressionIfTrue: This is the expression that gets executed if the condition is true.
  • expressionIfFalse: This is the expression that gets executed if the condition is false.

  • Example:-
    let number = 7;
    let result = (number % 2 === 0) ? "Even" : "Odd";
    console.log(`The number ${number} is ${result}.`);

    • Explain:-
  • This checks whether the remainder when number is divided by 2 is zero. If true, the number is even; if false, the number is odd.

    Usage of the Comma Operator in JavaScript :-

    The comma operator in JavaScript is used to evaluate multiple expressions in a single statement, with the final expression's result being returned. When using the comma operator, each expression is evaluated from left to right, but only the value of the last expression is returned.
    Syntax :- expression1, expression2, expression3, ... , expressionN
    Explain:-
  • expression1, expression2, ..., expressionN: These are the expressions to be evaluated. Each expression is evaluated in order, but only the result of the last expression (expressionN) is returned.

    • Example :-
    let x, y, z;
    x = (y = 10, z = 20);
    console.log(x); // Output: 20
    console.log(y); // Output: 10
    console.log(z); // Output: 20
  • In this example, y is assigned 10, z is assigned 20, and the value of z (20) is assigned to x.

    • Q3. Use a nested ternary operator to check that a number is positive, negative or zero. You have to print “positive” if the number is positive and similarly for negative and zero also.

    Using Nested Ternary Operator to Check if a Number is Positive, Negative, or Zero in JavaScript :-

    The ternary operator in JavaScript provides a concise way to perform conditional checks and execute expressions based on those conditions. A nested ternary operator is when one ternary operator is used inside another, allowing for multiple conditions to be evaluated sequentially.
    Ternary Operator Syntax:- condition ? expressionIfTrue : expressionIfFalse;
    Explain:-
  • condition: The condition to evaluate.
  • expressionIfTrue: Executed if the condition is true.
  • expressionIfFalse: Executed if the condition is false.

    • Example :-
    let numbers = 10;
    let numbers1=-1;
    let numbers2=0;
    let results = (numbers > 0) ? "positive" : (numbers < 0) ? "negative" : "zero";
    let results1 = (numbers1 > 0) ? "positive" : (numbers1 <0) ? "negative" : "zero";
    let results2 = (numbers2 > 0) ? "positive" : (numbers2 < 0) ? "negative" : "zero";
    console.log(`The number ${numbers} is ${results}.`);
    console.log(`The number ${numbers1} is ${results1}.`);
    console.log(`The number ${numbers2} is ${results2}.`);
    Explanation of the Code :-
  • declare a variable number and assign it the value 10 ,-1 ,0. We can change this value to test different cases.
  • First Condition: (number > 0) Checks if the number is greater than zero. If true, result is assigned "positive". If false, it moves to the next condition.
  • Second Condition: (number < 0) Checks if the number is less than zero. If true, result is assigned "negative". If false, result is assigned "zero".
